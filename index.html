What changed in this demo

1. Auto fuel spread: pilots enter “this stop” and “next stop” fuel prices; the calculator displays the difference automatically.

2. True economics: the tool always shows negative economics instead of hiding them, so tankering is only recommended when projected net savings are positive.

3. Per-leg GO / NO-GO: the tool recommends tankering whenever projected net savings are positive.

4. Landing cap and start fuel awareness: the output shows the landing fuel cap and uses max usable fuel / performance start fuel limits.

5. Actual validator sync: the actual savings checker pulls the first leg’s auto-calculated price spread.

6. Actual carry penalty: calculated as ((Actual Out − Actual In) − Planned Leg Fuel) ÷ (Actual Out − (Planned Leg Fuel + Non-Tanker Arrival Fuel)) ÷ Actual Flight Time × 100.

Numeric fields open the mobile number keypad on iPhone, iPad, and Android. This is not an FAA, dispatch, AFM, company-release, or performance-planning substitute. Always verify W&B, landing weight, runway analysis, reserves, MEL/CDL, fuel order requirements, and company SOPs.
